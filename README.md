# Pokédex Pokemon
Pokédex Pokemon created with CSS, React.js and uploaded to Firebase.

The list is from the [PokeApi](https://pokeapi.co/) ([JSON List](https://pokeapi.co/api/v2/pokemon?limit=3000&offset=0)).
